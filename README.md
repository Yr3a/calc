# Калькулятор оценки

Одностраничный сайт-калькулятор оценки. Работает полностью в браузере.

## Как выложить на GitHub Pages

1. Создай новый репозиторий на GitHub, например `grade-calculator`.
2. Загрузи в репозиторий файлы из этого архива: `index.html`, `.nojekyll`, `README.md`.
3. Открой `Settings` → `Pages`.
4. В разделе `Build and deployment` выбери `Source: Deploy from a branch`.
5. Выбери ветку `main` и папку `/root`, затем нажми `Save`.
6. Через пару минут сайт будет доступен по адресу:
   `https://ТВОЙ_ЛОГИН.github.io/grade-calculator/`
